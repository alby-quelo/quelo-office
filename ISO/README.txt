Quelo Office — ISO di pubblicazione
====================================

In questa cartella va distribuita UNA SOLA immagine:

  Quelo_Office-0.71-alpha.iso

È l'unica release alpha pubblicata. Le ISO di sviluppo intermedie (0.01–0.70)
non vanno caricate online né incluse nel repository pubblico.

prepare-usb.sh usa automaticamente la 0.71 (vedi QUELO_PUBLISH_ISO_VERSION).

Build successivi in locale (es. 0.72+) restano per uso interno finché non
decidi una nuova release pubblica.
